CLI

Quantizerにコマンドを送信したり、QMKのデバッグ出力を確認したりできます。

  • 送信したいコマンドを入力してEnterまたはSendボタンをクリック
    • ポップアップが表示された場合はQuantizerを選択

コマンド一覧

コマンド 説明
help 使用できるコマンド一覧を表示
debug on デバッグ出力を有効化
debug off デバッグ出力を無効化
dfu ブートローダを起動
ee save QMKでEEPROM上に書き込まれる設定(AG_SWAPなど)を永続化
